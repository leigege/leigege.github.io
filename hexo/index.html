买域名（可省略）

这一步骤可以省略，如果使用github的空间会自动分配以你的用户名为首的子域名。比如我的，bjlbeyond.github.io。看着有点借人篱下的感觉，所以我尝试申请了一个域名。
首选万网、国产大品牌、选择符合自己博客主题的域名，比如我的博客，希望以后更新一些自己的技术日常、安全相关、技术变现、普及一些我所知道的黑产相关的东西，故此选择了51heichan这个域名，花费55元（首年）。

环境配置

hexo
上有安装的详细教程、再次不再赘述、如有不懂留言即可
Node.js
这个是用来生成静态的页面、官网下载某个稳定版本、像平时安装软件、一路安装就行。

Git
作用：把本地hexo内容commit到GitHub上、因为GitHub提供的免费空间。如果是mac平台、xcode 自带git。windows需自行下载、一路安装。可以参考hexo官网的提示。

初始化

终端cd到一个你选定的目录，执行hexo init命令：
$ hexo init blog
blog是你建立的文件夹名称。cd到blog文件夹下，执行如下命令，安装npm：
$ npm installh
执行如下命令，开启hexo服务器：
$ hexo s

关联GitHub

添加ssh key到Github
执行如下命令生成public/private rsa key pair，注意将youremail@example.com换成你自己注册Github的邮箱地址。
$ ssh-keygen -t rsa -C "your_email@example.com"_
默认会在相应路径下（mac:/.ssh/id_rsa.pub）生成id_rsa和idrsa.pub两个文件。
前往文件夹/.ssh/idrsa.pub打开idrsa.pub文件，里面的信息即为SSH key，将这些信息复制到Github的Add SSH key页面即可。
进入Github –> Settings –> SSH keys –> add SSH key:
Title里任意添一个标题，将复制的内容粘贴到Key里，点击下方Add key绿色按钮即可。

github创建仓库

登录你的Github帐号，新建仓库，名为用户名.github.io固定写法，如bjlbeyond.github.io
终端cd到blog文件夹下，vim打开config.yml，命令如下：
$ vim _config.yml_

你需要将repository后bjlbeyond换成你自己的用户名

在blog文件夹目录下执行生成静态页面命令：
$ hexo generate 或者：hexo g
再执行配置命令：
$ hexo deploy 或者：hexo d
此时，浏览器中打开网址http://bjlbeyond.github.io

把bjlbeyond换成你的用户名
打开的页面和刚才本地一样

安装theme

你可以到Hexo官网主题页去搜寻自己喜欢的theme。这里以next主题为例
终端cd到 blog 目录下执行如下命令：
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
将blog目录下_config.yml里theme的名称landscape修改为next
终端cd到blog目录下执行如下命令(每次部署文章的步骤)：
$ hexo clean //清除缓存文件 (db.json) 和已生成的静态文件 (public)

$ hexo g             //生成缓存和静态文件

$ hexo d             //重新部署到服务器

至于更改theme内容，比如名称，描述，头像等去修改blog/_config.yml文件和blog/themes/next/_config.yml文件中对应的属性名称即可， 不要忘记冒号:后加空格。NexT 使用文档里有极详细的介绍。

在/blog/themes/landscape/source目录下新建文件名为：CNAME文件，注意没有后缀名！直接将自己的域名如：51heichan.com写入。
终端cd到blog目录下执行如下命令重新部署：
$ hexo clean //清除缓存文件 (db.json) 和已生成的静态文件 (public)

$ hexo g             //生成缓存和静态文件

$ hexo d             //重新部署到服务器

域名解析

如果将域名指向一个域名，实现与被指向域名相同的访问效果，需要增加CNAME记录。
登录你注册域名的网站，比如万网，在你购买的域名后边点击：解析 –> 添加解析
记录类型：CNAME
主机记录：将域名解析为example.com（不带www），填写@或者不填写
记录值：bjlbeyond.github.io. (不要忘记最后的.，bjlbeyond改为你自己的用户名)，点击保存即可。
完事了，这都不重要、重要的是博客的内容、以及你的坚持
